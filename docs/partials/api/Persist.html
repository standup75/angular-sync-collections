Persist

Description

Service used to retrieve collections from the database This service used to retrieve and persist objects from the Storage.

Dependencies

  • $q,

    $http, Config, Loader, Storage

Example

With a model

promise = Persist.init "humans", Human
promise.then -> humans = Persist.get "humans"

without default model

config = Persist.init "config"
config.perPage = 20
Persist.save config

delete everything

Persist.reset()
