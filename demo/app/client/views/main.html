Ok, so, what is the magic here? Open your network tab and reload the page: if you don't make any change the angular-sync-collection module will only request the counter for that collection, it sees that the counter has not changed, so it does not need to request the full collection again

But if you perfomr one the following CRUD operation, you will see that the counter has changed and it will request the full collection again
